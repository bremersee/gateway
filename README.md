# Bremersee's Spring Cloud Gateway

This gateway is more or less a plain Spring Cloud Gateway:

```xml
<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-starter-gateway</artifactId>
</dependency>
```
